This is Sample Page of Papago-Now

'Papago-Now' is translator, that translate text through the Naver Papago. (Papago is very nice translator in the Korea)
Also, 'Papago-Now' translate on the Puppeteer. Due to this, 'Papago-Now' is very simple and very slow(...).

Usage

    document.querySelector('form').addEventListener('submit', (e) => {
        e.preventDefault();
        let data = {
            so: 'ko',
            ta: 'en',
            text: e.target.text.value
        }
        // fetch('https://papago-now.herokuapp.com/translate', {
        fetch('/translate', {
            method: "POST",
            headers: {'Content-Type':'application/json'},
            body: JSON.stringify(data),
        })
        .then(res => res.json())
        .then(json => {
            document.querySelector('h1').innerText = json.result
        });
    })
